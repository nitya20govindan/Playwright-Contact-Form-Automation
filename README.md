# Playwright Contact Form Automation

## Overview
This repository contains test automation for a contact form using Playwright. The tests cover both positive and negative scenarios for the form.

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nitya20govindan/Playwright-Contact-Form-Automation.git
   cd Playwright-Contact-Form-Automation
   ```

2. **Install dependencies:**
   Ensure you have [Node.js](https://nodejs.org/) installed. Then run:
   ```bash
   npm install
   ```

3. **Run the tests:**
   To run all tests:
   ```bash
   npx playwright test
   ```

   For individual tests, you can specify the test file:
   ```bash
   npx playwright test tests/contact.spec.js
   ```

4. **Test Reports:**
   Test results will be generated in the `test-results/` folder. You can inspect them after each test run.
